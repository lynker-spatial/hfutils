# Get an OAuth2 token for Lynker Spatial

Get an OAuth2 token for Lynker Spatial

## Usage

``` r
lynker_spatial_token(..., client = lynker_spatial_client())
```

## Arguments

- ...:

  Unused; reserved for future options.

- client:

  An OIDC client, as returned by \[lynker_spatial_client()\].

## Value

An httr2 OAuth2 token.
