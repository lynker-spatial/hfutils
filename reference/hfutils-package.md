# hfutils: Core Utilities for the Hydrofabric Stack

Base-layer utilities for accessing, cleaning, summarizing, and
versioning hydrofabric datasets. Provides hydrofabric I/O (GeoPackage,
GeoParquet, and lazy GDAL/OGR-SQL access), DEM-derived geometry
cleaning, network and topology algorithms (downstream accumulation,
hydrosequence), staged pipeline invariant checks, and GeoPackage
dataset-version and provenance metadata helpers used across the
hydrofabric package suite.
